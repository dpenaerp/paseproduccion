# Proyecto de Registro de Pases a Producción

Este proyecto es una aplicación web en Flask para registrar y visualizar pases a producción. Permite registrar información detallada sobre cada pase y almacenarla en una base de datos SQLite.

## Estructura del Proyecto

- **app.py**: Código principal de la aplicación en Flask.
- **templates/**: Carpeta con las plantillas HTML para la interfaz web.
- **pases_produccion.db**: Base de datos SQLite para almacenar los pases a producción. (Se ignora en `.gitignore`).

## Requisitos

- Python 3.x
- Flask (`pip install flask`)

## Uso

1. Clona el repositorio.
2. Instala las dependencias: `pip install -r requirements.txt`.
3. Ejecuta la aplicación: `python app.py`.
4. Accede a la aplicación en `http://127.0.0.1:5000/`.
